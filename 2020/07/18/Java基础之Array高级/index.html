Java基础之Array高级

#《Java基础之Array高级》

学习总结

###（一）冒泡排序

  • 依次找出最大值，次大值，第一次排序结束后数组最后一个值为最大值
  • j 与 j-1 比较

###（二）选择排序

  • 找出最小值，并放在数组的第一个位置
  • i 与 j 进行比较

###（三）二分法查找

  • 数组要先排序，才能进行二分法查找

###（四）Arrays类

  • public static String toString(int[] a) 把数组转成字符串
  • public static void sort(int[] a) 对数组进行排序
  • public static int binarySearch(int[] a,int key) 二分查找
